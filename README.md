# TodoListPlus

This app is used to allow users to create a list of tasks for things that they need or want to do.

To create a task the user types in the name of the task in the text box provided and then clicks the add button, the task will then appear on screen with the coordinates of the location of where the task was entered. The user can then edit a task, take a photo, view the photo or delete a task. 

To edit a task, the user can click on the "Edit" button, this will bring up a text box for the user to type in a new name for the task, they can then click the "Save" button to save the new name for the task. 

To take a photo, the user can click on the "Take Photo" button, this will open the devices camera, the user can then click the "Capture Photo" button, this will take the picture and show it on the screen, the user can then click the "Save Photo" button to save the photo. 

The user can view the photo by clicking the "View Photo" button, this will allow them to see the photo that they took (for this app, only the first taken photo for each task will show). To delete a task, the user can click the "Delete" button, this removes the task from the list.
